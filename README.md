File Organizer using Python



A Python automation script that organizes files into categorized folders based on file extensions. The program uses the os and shutil modules to automatically create folders and move files, resulting in a clean and well-structured directory.



Features:-



* Automatically sorts files by type
* Creates folders if they do not exist
* Supports Images, Documents, Videos, Audio, and Archives
* Handles uncategorized files using an "Others" folder
* Simple command-line interface



Technologies Used:-



* Python
* os module
* shutil module
